# Lets Go
A sample who will deal with GTFS file
- Working on vim 
- Plugin vim-go https://github.com/fatih/vim-go

## Implementation
Steps : 
- 1. Create a sqlite database
- 2. Parse some text file
- 3. Fill the db
- 4. Start a gin server
- 5. Let one URL available for get the data's
